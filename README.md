# 💳 Bank Account Management System (C++)

## 📌 Overview

This project is a **Bank Account Management System** developed in C++.
It is a menu-driven program that allows users to create, manage, and store bank account data efficiently.

The system demonstrates core programming concepts such as **Object-Oriented Programming (OOP)**, **file handling**, and **dynamic memory management**.

---

## 🚀 Features

* ➕ Create new bank accounts
* 💰 Deposit money into an account
* 💸 Withdraw money with balance validation
* 📄 Display all account details
* ❌ Delete an account (with array adjustment)
* 💾 Save account data to file (`accounts.txt`)
* 🔄 Automatically load data from file on startup
* 📊 Track number of current and saving accounts (static members)

---

## 🛠️ Technologies & Concepts Used

* C++ Programming
* Classes & Objects
* Constructors (Default & Parameterized)
* Enumerations (Account Type: Current/Saving)
* Static Data Members & Functions
* File Handling (`ifstream`, `ofstream`)
* Dynamic Memory Allocation (Pointers)
* Menu-Driven Program Design
